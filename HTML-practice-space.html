CSS Positioning

Static: Boxes are what they are.

Relative: boxes can be nudged relative to their static position. E.g. push it up, down, left, right by a certain number of pixels from where it otherwise would have been.

Absolute: position relative to nearest position ancestor/parent (e.g. when it is nested). So if parent box has a lower bound at 400px,

Fixed: positioned to exact coordinates in the browser window. This is the most sensible way. Positions are not inherited from ancestors.
